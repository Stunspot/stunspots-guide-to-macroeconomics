# Status — Stunspot’s Guide to Macroeconomics

*Stunspot’s Guide to Macroeconomics* is a **v1.0 public canon release** built from an existing model-authored macroeconomics knowledge corpus.

The release is structurally complete for its intended purpose: AI/RAG ingestion, project knowledge upload, source-level browsing, and public reference.

## Release State

- Version: **1.0**
- Release date: **2026-06-27**
- Source reports: **13**
- Compiled packs: **4**
- Omnibus files: **1**
- License: **Creative Commons Attribution 4.0 International**
- Primary audience: **AI/RAG systems and model-facing workflows**

## Corpus Maturity

The source reports are the canonical individual units. Compiled packs and the omnibus are convenience formats generated from the report corpus.

This repository is ready for:

- AI project-knowledge upload
- RAG corpus ingestion
- long-context research sessions
- source-level citation and review
- macro-financial reasoning support
- institutional strategy and diagnostic workflow support

## Directory Policy

- `docs/` is navigation and guidance.
- `knowledge-packs/by-report/` contains the 13 individual source reports.
- `knowledge-packs/compiled-packs/` contains the 4 grouped upload packs.
- `knowledge-packs/omnibus/` contains the 1 whole-corpus bundle.
- `docs/reports/` is intentionally not used.

## Known Boundaries

The canon supplies conceptual structure, terminology, causal models, and diagnostic scaffolding. Current-event applications require fresh data and source verification.
