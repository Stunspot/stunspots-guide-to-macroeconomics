## Maturity

This is a complete first public release of the canon package. The repository includes the canonical individual reports, grouped upload packs, whole-corpus omnibus, citation metadata, release manifests, and public navigation docs.

The release is mature enough for AI/RAG ingestion, project knowledge attachment, long-context reference use, local search, and source-level inspection.

It should not be treated as a live macroeconomic data feed, investment recommendation system, legal/financial advice engine, or peer-reviewed academic publication. Verify current data, policy decisions, market prices, laws, and high-impact claims before relying on them.

## Directory Policy

- `docs/` contains navigation, GitHub Pages metadata, and usage guidance.
- `knowledge-packs/by-report/` contains the canonical individual source reports.
- `knowledge-packs/compiled-packs/` contains grouped upload packs.
- `knowledge-packs/omnibus/` contains the whole-corpus bundle.

No source reports are duplicated under `docs/`.

## Canonical Units

The source reports in `knowledge-packs/by-report/` are the canonical individual units. Compiled packs and omnibus files are convenience formats for upload, archive, and long-context use.
